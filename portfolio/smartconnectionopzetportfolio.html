Gebruik Bootstrap

Als eerste besloot ik om te zoeken naar een front-end framework. Er zijn verschillende frameworks zoals Bootstrap en Foundation. Een framework zorgt ervoor dat je zelf weinig tot geen css hoeft te maken maar enkel classes op moet roepen om de HTML te stylen. Ik heb ervoor gekozen om Bootstrap te gebruiken omdat daar veel documentatie over is terug te vinden. Daarnaast is het gratis en Open Source. Dus geen moeilijkheden met licenties.

Responsive programmeren

Probeer het eens uit om mijn portfolio te openen op je mobiele telefoon. Tijdens het ontwikkelen van mijn website heb ik er alle moeite voor gedaan om hem responsive te maken. Dit betekent dat de website meeschaalt zodra het venster groter of kleiner wordt. Door Bootstrap goed te gebruiken is mij dit gelukt.

Programmeertalen

Voor dit portfolio heb ik gebruik gemaakt van verschillende programmeertalen. Voor de opmaak zijn dat dus HTML en CSS. Maar ik heb ook JavaScript gebruikt. Met name voor de verschillende API's, AI, en het gastenboek. Ook heb ik gebruik gemaakt van een NoSQL Database. Hierover heb ik een video gemaakt. Deze is hier te zien.



Structuur Portfolio

Ik heb eerst goed nagedacht hoe mijn portfolio overzichtelijk te gebruiken zou zijn. Ik vond met name de navigatie belangrijk. De verschillende knoppen moesten duidelijk zijn en de bezoekers moeten niet "verdwaald raken" op mijn website. Door verschillende manieren van navigeren en verschillende pagina's te gebruiken heb ik ervoor gezorgd dat dit niet gebeurt.
