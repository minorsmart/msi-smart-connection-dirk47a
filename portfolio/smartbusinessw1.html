User profiles en value mapping

De opdracht die ik tijdens deze eerste les van Smart Business kreeg was om een eigen user profile te maken en een Smart mapping van Smart Industry. Ik heb tijdens deze les overlegd met medestudenten die al vaker een zelfde soort opdracht hadden gemaakt omdat dit compleet nieuw was.
Hier links zie je de Value Map. Dit is in feite het "Cadeautje" dat de Minor Smart Industry ons geeft. Maar niet alle mensen vinden dezelfde cadeautjes leuk. Het moet uiteraard wel bij iemand passen. Hiervoor is de persona profile (rechts in beeld). De persona profile is de deelnemer van de minor zelf. (ik in dit geval). Door dit in te vullen kun je erachter komen of de Minor Smart Industry wel bij jou past. Je kunt antwoord krijgen op de volgende vragen:

De afbeelding hieronder laat zien hoe ik zelf deze opdracht heb ingevuld. Ook heb ik dit model erbij gepakt op het moment dat ik voor Smart Start mijn persoonlijke SWOT-analyse maakte.
